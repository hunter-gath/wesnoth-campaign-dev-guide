# Hunter Gath

Hunter Gath is a compact, two-scenario Battle for Wesnoth campaign intended as a campaign-development sample.

Install it by extracting this `Hunter_Gath` folder into your Wesnoth user add-ons directory, usually:

```text
~/.local/share/wesnoth/<version>/data/add-ons/Hunter_Gath
```

Run `wesnoth --config-path` to confirm your exact userdata path.

The placeholder art in this package was generated for scaffolding only. Replace it before publishing.
