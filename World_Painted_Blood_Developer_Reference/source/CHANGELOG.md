# Changelog

## 0.1.0 — Initial full test build

- Added complete 20-scenario campaign chain.
- Added 20 distinct underground maps.
- Added custom Hunter unit.
- Added persistent rescue, sacrifice, corruption, truth, and alliance systems.
- Added one explicit moral choice and one unique reinforcement event per scenario.
- Added multi-objective and multi-leader victory tracking.
- Added final escape phase and two ending tones.
- Added silent placeholder OGG files for all planned custom music.
- Added README, sound manifest, implementation notes, and structural validator.
- Disabled default leader-elimination victory so every scenario obeys its custom objective state.
- Changed the Hunter to inherit the core Ranger's complete unit assets and animations.
- Added explicit Hunter-death defeat handling and safe nonlethal bait damage bookkeeping.
- Expanded validation to check coordinate reachability, sound references, custom music linkage, and Ogg decoding.
