Credits and status

The campaign credits Bryant Family for concept, design, and narrative. The implementation is labeled as a generated development build intended for iterative playtesting.

The metadata description explicitly calls this an “Initial test build for Wesnoth 1.18.” Keep that wording until an engine playthrough and balance pass justify a stronger release claim.
