Image boundary

Version 0.1.0 does not ship campaign-local image art. The campaign menu and Hunter inherit core unit images. Future portraits, story panels, icons, and custom unit sprites should be added under a campaign-local images/ directory without changing the fixed scenario, map, or music filenames.
