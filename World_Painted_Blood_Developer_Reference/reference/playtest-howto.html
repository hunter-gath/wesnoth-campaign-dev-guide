Install the add-on

  1. Copy the complete World_Painted_Blood/ folder into the active Wesnoth 1.18 user-data data/add-ons/ directory.
  2. Confirm that _main.cfg is directly inside World_Painted_Blood/.
  3. Start Wesnoth or press F5 at the title screen to reload add-ons.
  4. Choose World Painted Blood from Campaigns.

Fast scenario testing

During development, enable debug mode and use :cl to select a scenario. An alternative is temporarily changing first_scenario in _main.cfg; revert that change before packaging.

Minimum pass for each scenario

AreaQuestions
LoadDoes the scenario enter without WML errors? Do the map and custom music filename resolve?
ObjectivesDo all labels exist, counters update exactly once, and victory occur in either completion order?
Moral choiceDo both options work, write the intended variables, and leave no unusable unit state?
AI and economyCan enemies recruit? Are turns, gold, income, and reinforcement timing reasonable on all difficulties?
TransitionDoes victory load the exact next scenario and preserve the recall list and campaign variables?
DefeatDoes Hunter death and time expiry behave correctly?

Run the bundled validator

cd World_Painted_Blood
python3 tools/validate_wpb.py

This checks filenames, chain integrity, map references, music references, sound basenames, tag balance, terrain codes, coordinates, reachability, and archive assumptions. It cannot replace an engine run.
