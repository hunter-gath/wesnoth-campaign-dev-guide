Event composition pattern

Most scenarios use the same readable spine: prestart, start, one moral-choice moveto, one moveto per objective site, one die event per required leader, and one timed reinforcement event. Scenario 09 adds a campaign-menu ability; Scenario 20 adds phase-change and escape events.
