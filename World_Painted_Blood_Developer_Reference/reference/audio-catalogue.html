Replacement contract

Do not rename the music files. Replace each silent placeholder in place. The scenario WML already points to 01_Scenario.ogg through 20_Scenario.ogg.

Sound-copy plan

Version 0.1.0 references installed core sounds by basename. The bundled sound manifest records their expected source paths so campaign-local copies can be added later without changing call sites.
