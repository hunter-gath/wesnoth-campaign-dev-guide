Fresh versus reused production

Content needOpening arc’s approachImplication for a new campaign
Terrain graphicsUses core terrain codes and graphics.Custom map tiles are not required for a distinctive campaign.
MapsCampaign-specific map data assembled from core terrain.You usually create maps, not terrain art.
Hero portraitsCampaign-specific for Haldric, Eldaric, and Burin.Original portraits give the highest identity return.
Standard faction portraitsReuses orc/outlaw portrait paths where suitable.Not every speaking leader needs new art.
MusicReuses the shared Wesnoth score.Original music is optional, not a baseline requirement.
Unit spritesCustom for Haldric/Wesfolk lines; core for most armies and Burin.Create custom sprites only where the unit identity or mechanics justify them.
UI and sceneryReuses icons, signposts, and standard composition assets.Core presentation assets cover many interaction needs.
