Unit authoring patterns

PatternExampleLesson
Full custom progression lineNoble Fighter → Noble Commander → Noble LordUse when the protagonist’s mechanics and visual identity are central.
Custom unit using standard movement type, attacks, icons, and soundsNoble Fighter / CommanderCustom units can still reuse most engine and core assets.
Core base-unit wrapperWarrior King → Royal WarriorPrefer inheritance when only story role or a few abilities differ.
Named core unitBurin the Lost → Dwarvish SteelcladA memorable hero does not require a custom unit type.
