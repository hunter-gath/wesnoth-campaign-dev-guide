State-minimization principle. A narrative fact does not always need a permanent variable. Once all valid campaign paths guarantee the same fact, later code can treat it as an invariant.

Ruby concealment

StageMechanical representation
Scenario 12Haldric receives permanent object ruby_of_fire.
Scenario 19[remove_object] object_id=ruby_of_fire removes the attack from Haldric.
Scenario 19Jessene leaves the battlefield to hide the physical artifact.
Scenario 20Jevyan detects only that the Ruby’s magical power has been concealed.
Artifact-model lesson. A permanent object can later be removed just as explicitly as it was added. The unit’s attack list is part of campaign state.
